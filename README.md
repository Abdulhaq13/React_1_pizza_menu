Pizza Menu 🍕
A simple React application that displays a pizza restaurant menu. This project was built while learning React fundamentals such as components, props, and conditional rendering.

Features
Component-based UI
Dynamic rendering of menu items
Conditional rendering for sold-out items
Clean and reusable components
Technologies Used
React
JavaScript
HTML
CSS
Project Structure
src/ components/ App.js index.js data.js styles.css


The application will run on http://localhost:3000

What I Learned
Creating and structuring React components
Passing data using props
Rendering lists in React
Writing cleaner and reusable UI components

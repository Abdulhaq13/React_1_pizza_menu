# 🍕 Pizza Menu App

A simple React application that displays a pizza menu with different items, built while learning React fundamentals.

This project is part of the **Ultimate React Course by Jonas Schmedtmann**, focusing on understanding core React concepts through hands-on practice.

---

## 🚀 Features

* 📋 Display a list of pizzas
* 🍕 Show pizza name, ingredients, and price
* ❌ Indicate when a pizza is sold out
* 🧩 Component-based UI structure
* ⚡ Fast and clean user interface

---

## 🛠️ Tech Stack

* React (Functional Components)
* JavaScript (ES6+)
* CSS

---

## 📚 Concepts Learned

This project helped me understand:

* React Components
* JSX (JavaScript XML)
* Props (Passing data between components)
* Conditional Rendering
* Rendering Lists using `.map()`
* Component reusability

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Abdulhaq13/React_1_pizza_menu.git
```

### 2. Navigate to the project folder

```bash
cd React_1_pizza_menu
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm start
```

Open 👉 http://localhost:3000
